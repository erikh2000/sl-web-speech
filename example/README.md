# Example App

Just a minimal app to test out the library.

## Installation

In the parent directory (`sl-web-speech`)...
1. `npm install`
2. `npm run build`
3. `npm link`

In this directory (`example`)... 
4. `npm install`
5. `npm link sl-web-speech`
6. `npm run build`
7. `npm run start`

The last step should open a web browser and load the app.
